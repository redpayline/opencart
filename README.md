RED PAY LINE Payment Module for OpenCart ****
INSTALLATION MANUAL

1. Download and unpack the module
2. Register on the https://www.redpayline.com

6. Copy the folders of module: Admin and Catalog from the downloaded files into system files of OpenCart
7. Go to the "admin" panel of OpenCart, go to the "Extensions / Payment"
8. Select CRedpayline and click [Install]
9. Select Redpayline and click [Edit]

10. Fill in the fields in the dialog box:
	- User Name					- You got it in the account of https://www.redpayline.com
	- Status 					- «Enabled»
	- Completed status 			– «processed»
	- Failed Status 			– «failed»
	- Geo Zone 					- «All Zones»
	- Sort Order 				- 1
11. Then click «Save»
